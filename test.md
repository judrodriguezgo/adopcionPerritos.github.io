# COMMAND LINE

## Most useful commands

| Command | Description |
| ------- | ----------- |
| `which <COMMAND>` | Busca la ubicacion de un comando en la variable PATH. |
| `type <COMMAND>` | Determina información acerca de un comando. |
| `type -a <COMMAND>` | Determina información acerca de un comando |
| `type -t <COMMAND>` | Mostrar una sola palabra que describe al comando. |
